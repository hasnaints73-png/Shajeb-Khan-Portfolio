# Shajeb Khan | Full-Stack Developer Portfolio

A responsive, high-performance personal portfolio built to showcase MERN stack expertise, academic background, and upcoming projects. 

## 🚀 Overview

This repository contains the front-end source code for my personal developer portfolio. Currently studying Electronics & Communication Engineering (ECE) at Jamia Millia Islamia (3rd Year), I am transitioning into software engineering with a core focus on the MERN stack.

### Core Technologies
- **Structure:** HTML5 Semantic Markup
- **Styling:** Custom CSS3 with Modern CSS Variables, Flexbox, & Display Grid
- **Interactivity:** Vanilla JavaScript (ES6+) utilizing native Intersection Observers

## ✨ Features

- **Modern UI/UX:** Clean, dark-mode design system with neon accents to emulate a terminal hacking aesthetic.
- **Scroll Animations:** Native DOM Intersection Observers triggering staggered, buttery-smooth CSS keyframes in real-time.
- **Responsive Design:** Mobile-first fluid typography and layout rules ensuring perfect rendering everywhere from mobile devices to ultrawide monitors.
- **Fast Performance:** Zero heavy frameworks or libraries, boasting an ultra-light JS payload and sub-second load times.

## 🛠️ Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/hasnaints73-png/Shajeb-Khan-Portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd Shajeb-Khan-Portfolio
   ```
3. Run it using any local HTTP server (such as the VS Code Live Server extension) or simply double-click `index.html` to open it in your browser.
